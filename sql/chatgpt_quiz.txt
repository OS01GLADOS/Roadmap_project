- "Что означает SQL?" - a) Structured Query Language (язык структурированных запросов).
- Уникальное ключевое слово для идентификации каждой записи в таблице - это PRIMARY KEY, а не UNIQUE KEY.
- Для удаления колонки из существующей таблицы в SQL используется оператор ALTER TABLE с командой DROP COLUMN.
- В большинстве баз данных, таких как MySQL и PostgreSQL, используется оператор LIMIT для ограничения 
	количества возвращаемых строк.
	Однако в некоторых других базах данных, таких как Microsoft SQL Server, может использоваться оператор TOP.
- SQL-оператор JOIN используется для объединения результатов из двух или более таблиц,
	но для объединения результатов двух или более запросов в один результат используется оператор UNION.
- Для добавления новой колонки в существующую таблицу SQL использует оператор ALTER TABLE с командой ADD COLUMN.
- Внутреннее соединение (INNER JOIN) возвращает только строки, которые имеют совпадения в обеих таблицах,тогда 
	как внешнее соединение (OUTER JOIN) возвращает все строки из одной таблицы и совпадающие строки из другой.
- Агрегатные функции в SQL - это функции, которые применяются к столбцам данных и возвращают одно значение,
	например, сумму, среднее значение, максимум или минимум.
- Внутреннее соединение (INNER JOIN) возвращает только совпадающие строки из обеих таблиц, тогда как внешнее 
	соединение (OUTER JOIN) возвращает все строки из одной таблицы и совпадающие строки из другой.
- Какие операторы SQL используются для группировки данных и агрегации значений в SQL запросах? - GROUP BY и HAVING
- виды индексов Одноколоночный индекс (SINGLE-COLUMN INDEX) 
	и многоколоночный индекс (MULTI-COLUMN INDEX) - отличаются количеством индексируемых столбцов.
